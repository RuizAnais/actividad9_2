Border-top

Border-top: Establece algunas o todas las propiedades del borde superior de los elementos (anchura,estilo de borde y color).


Border-bottom

La propiedad border-bottom se utiliza para establecer de forma abreviada el valor de una o más propiedades simultáneamente, como el grosor, estilo y color del borde inferior de un elemento.


Margin

Establece la anchura de algunos o todos los márgenes de los elementos. La propiedad margin es tan poderosa que permite establecer uno, dos, tres o los cuatro márgenes de forma simultánea. En este ejemplo se indican tres valores:

  1. margin-top: hace referencia al márgen superior.
  2. margin-right: el valor del márgen horizontal derecho.
  3. margin-bottom: es el margen inferior.
  4. margin-left: el valor del márgen horizontal izquierdo.


Padding

Establece la anchura de algunas o todas las zonas de relleno de los elementos. En este ejemplo se indican cuatro valores:

  1. padding-top: es el relleno superior.
  2. padding-right: es el relleno derecho.
  3. padding-bottom: es el relleno inferior.
  4. padding-left: es el relleno izquierdo.


Vertical align

La propiedad vertical-align puede usarse para alterar la ubicación o posicionamiento vertical de un elemento en línea, en relación a su elemento padre o a la línea del elemento. (Un elemento en línea es uno que no tiene salto de línea ni antes ni después de él).

Text-indent (Sangría de texto)

La propiedad text-indent indica la tabulación o indentación que se aplica a la primera línea de texto de un elemento.

El tamaño de la tabulación se puede indicar con cualquiera de las unidades de medida definidas por CSS, puede aplicarse a elementos de nivel bloque (P, H1, etc.) Puede ser positivo o negativo, mediante cualquiera de las unidades de medida incluyendo los porcentajes.


Word-wrap

La propiedad Word-wrap sirve para controlar palabras largas que pueden exceder un contenedor, podemos hacer que la palabra larga se interrumpa y continúe en la siguiente línea, de tal manera que queda ajustada dentro del contenedor o podemos dejar tal cual, para ello haremos uso de un par de valores

El valor normal hace que la no se altere y quede como por defecto, es decir no hace efecto visible sobre el texto; si el texto sobrepasa el contendor, entonces continuará sobrepasando con el valor normal.

El valor break-word hará que las palabras largas que sobrepasa un contenedor se ajuste y quede totalmente dentro de dicho contenedor, para ello se romperán dichas palabras si es necesario para continuar en la siguiente línea de texto.
